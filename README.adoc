= Cloud Native Zuul Server

A small Spring Cloud Zuul server application used for demos and showcases.

== Build instructions

In order to compile and run the examples you do not need much.
```shell
$ ./gradlew clean build
```

== Running the Cloud Native Zuul Server

```shell
$ ./gradlew bootRun
```
